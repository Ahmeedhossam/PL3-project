4. Dictionary Application
Objective:
Create a functional F# dictionary with full CRUD and search features.

Description:
Students develop a digital dictionary stored in a Map. Focus on efficient searches, updates, 
and persistence.

Features:
• Add/update/delete words
• Case-insensitive search
• Partial match search
• Save/load JSON or XML

Team Roles:
1. Word Model Developer
2. CRUD Developer
3. Search Engine Developer
4. File I/O Developer
5. UI Developer
6. Error Handling Developer
7. Tester
8. GitHub Manager
